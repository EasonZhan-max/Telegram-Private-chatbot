# Telegram 机器人主页简介文案

这些文案可以放到 BotFather 里设置机器人资料。

## 一、机器人名称 Bot Name

```txt
个人博客留言机器人
```

## 二、机器人用户名 Username

用户名需要自己在 BotFather 创建，必须以 `bot` 结尾。

示例：

```txt
my_blog_message_bot
contact_me_bot
blog_contact_robot
```

## 三、About 简短简介

适合放在 Telegram 机器人资料页的短简介里。

```txt
一个用于个人博客的 Telegram 留言机器人，支持私聊消息转发与管理员双向回复。
```

更短版本：

```txt
个人博客留言入口，支持 Telegram 双向私信回复。
```

## 四、Description 详细介绍

适合放在 Telegram 机器人主页介绍里。

```txt
欢迎使用个人博客留言机器人。

你可以直接发送文字、图片、视频、语音或文件，机器人会自动转交给管理员。
管理员回复后，你会在这里收到回复。

适合留言、反馈、合作联系、问题咨询等场景。
```

## 五、Commands 命令列表

可以在 BotFather 的 `/setcommands` 里填写：

```txt
start - 开始使用机器人
help - 查看使用说明
id - 查看自己的 Telegram 数字 ID
```

## 六、博客按钮文案

适合放在个人博客的联系按钮上：

```txt
通过 Telegram 联系我
```

```txt
给我留言
```

```txt
Telegram 私信留言
```
