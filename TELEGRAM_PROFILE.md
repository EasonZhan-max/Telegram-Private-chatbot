# Telegram 个人资料 Bio 文案

这些文案是给你放到 **Telegram 个人资料简介 Bio** 里的。

把 `your_contact_bot` 换成你自己的机器人用户名。

---

## 一、最推荐版本

适合直接放 Telegram 个人资料简介：

```txt
联系请走机器人留言：https://t.me/your_contact_bot
```

---

## 二、适合“主号不方便主动私信”场景

```txt
如果我无法主动私信你，请在这里留言：https://t.me/your_contact_bot
```

```txt
主号不方便主动私信，有事请通过这里联系：https://t.me/your_contact_bot
```

```txt
联系入口：https://t.me/your_contact_bot
```

---

## 三、短一点的版本

```txt
私信留言：https://t.me/your_contact_bot
```

```txt
有事找我：https://t.me/your_contact_bot
```

```txt
留言入口：https://t.me/your_contact_bot
```

---

## 四、比较正式的版本

```txt
合作 / 反馈 / 咨询请通过这里联系我：https://t.me/your_contact_bot
```

```txt
不常看陌生私信，有事请通过机器人留言：https://t.me/your_contact_bot
```

---

## 五、比较隐私的版本

```txt
不公开主号私信，有事请留言：https://t.me/your_contact_bot
```

```txt
临时联系入口：https://t.me/your_contact_bot
```

---

# BotFather 机器人主页资料文案

下面这些是放到 BotFather 里设置机器人资料用的。

---

## Bot Name 机器人名称

```txt
双向联系中转机器人
```

也可以用：

```txt
私信留言机器人
```

```txt
个人联系机器人
```

```txt
联系中转机器人
```

---

## About 简短简介

```txt
用于 Telegram 个人资料 Bio 的双向联系中转机器人。
```

更短版本：

```txt
Telegram 私信留言入口。
```

---

## Description 详细介绍

```txt
欢迎使用双向联系中转机器人。

你可以主动发送文字、图片、视频、语音或文件，消息会自动转交给本人。
本人回复后，你会在这里收到回复。

说明：本机器人只作为被动联系入口，不会主动私信陌生用户。
```

---

## Commands 命令列表

可以在 BotFather 的 `/setcommands` 里填写：

```txt
start - 开始使用机器人
help - 查看使用说明
id - 查看自己的 Telegram 数字 ID
```
